# Introduction to Phishing

## Scenario Overview

This SOC Simulator exercise focused on identifying and investigating a phishing email alert generated by the security monitoring platform.

The email claimed that unusual sign-in activity had been detected on a Microsoft account and urged the recipient to click a link to review account activity.

## Investigation Process

The alert was reviewed to determine whether the email represented a legitimate notification or a phishing attempt.

The following indicators were identified:

* Suspicious sender domain impersonating Microsoft
* Urgent language designed to create panic
* External link directing the user to a non-Microsoft domain
* Mismatch between the sender address and legitimate Microsoft infrastructure
* Potential credential harvesting attempt

## Findings

The email was determined to be malicious and designed to trick users into submitting account credentials through a fraudulent website.

The sender domain was not owned by Microsoft and the message contained characteristics commonly associated with phishing campaigns.

## Classification

True Positive

## Escalation

Yes

## Outcome

The alert was escalated for further investigation and containment. Users should be warned not to interact with the email, and the sender domain should be blocked where appropriate.
