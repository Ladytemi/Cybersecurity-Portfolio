# Screenshot Evidence

This folder contains screenshots collected during the investigation of a Microsoft account phishing email.

Evidence includes:

1. Initial alert details showing sender, recipient, subject, and suspicious email content.
2. Incident report Part 1 showing classification, affected entities, and phishing rationale.
3. Incident report Part 2 showing escalation reasoning and remediation actions.
4. Incident report Part 3 showing attack indicators and final findings.

The screenshots support the conclusion that the email was a confirmed phishing attempt and was correctly classified as a True Positive.
