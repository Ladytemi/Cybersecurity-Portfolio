# Splunk 101

## Overview

Completed the Splunk 101 room and gained foundational knowledge of Splunk Enterprise, including data onboarding, indexing concepts, source types, and navigation of the Splunk interface.

## Skills Demonstrated

- SIEM Fundamentals
- Splunk Enterprise Navigation
- Data Ingestion Concepts
- Source Type Identification
- Index Management Concepts
- Log Management
- Security Monitoring Fundamentals
- Data Onboarding Workflow
- Splunk Architecture Awareness

## Topics Covered

1. Introduction to Splunk Enterprise
2. Splunk User Interface
3. Data Ingestion Methods
4. Source Types
5. Host Identification
6. Index Configuration
7. Data Onboarding Workflow
8. Splunk Applications and Dashboards

## Key Learning Outcomes

- Explored the Splunk Enterprise interface.
- Reviewed methods for onboarding and monitoring data.
- Learned how Splunk organizes data using hosts, source types, and indexes.
- Examined the workflow used when adding data into Splunk.
- Reviewed available Splunk applications and dashboards.

## Outcome

Developed foundational Splunk SIEM knowledge and understanding of data ingestion, indexing, and platform navigation used in Security Operations Centers (SOCs).
