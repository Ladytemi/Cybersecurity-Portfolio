## Overview

This room introduces the MITRE ATT&CK Framework, a globally recognized knowledge base used to classify adversary tactics, techniques, and procedures (TTPs).

MITRE ATT&CK provides security teams with a structured way to understand attacker behavior, map security controls, perform threat hunting, and improve detection capabilities.

The framework is widely used by SOC analysts, incident responders, threat hunters, red teams, blue teams, and cybersecurity professionals.

## Key Concepts Learned

### Tactics

High-level adversary objectives during an attack.

Examples:

- Initial Access
- Execution
- Persistence
- Privilege Escalation
- Defense Evasion
- Credential Access
- Discovery
- Lateral Movement
- Collection
- Exfiltration
- Impact

### Techniques

Specific methods used by attackers to accomplish a tactic.

Example:

- Phishing
- PowerShell
- Credential Dumping
- Remote Services

### Procedures

Real-world implementation of techniques by threat actors.

## MITRE D3FEND

MITRE D3FEND is a defensive knowledge graph that complements the ATT&CK framework. While ATT&CK focuses on adversary behaviors and techniques, D3FEND maps defensive countermeasures and security controls that can be used to detect, prevent, or mitigate those techniques.

Together, ATT&CK and D3FEND help security teams understand both offensive and defensive perspectives of cybersecurity operations.

## Key Takeaway

MITRE ATT&CK is one of the most important frameworks in modern cybersecurity and serves as a common language for describing and analyzing adversary behavior.
