# Junior Security Analyst Intro

## Date Completed

June 2026

## Objective

Understand the responsibilities of a Junior Security Analyst within a Security Operations Centre (SOC) environment and gain exposure to key cybersecurity functions including threat intelligence, digital forensics, and incident response.

## Task 1: Security Operations Centre (SOC)

### Key Concepts Learned

* Purpose and functions of a Security Operations Centre (SOC)
* Responsibilities of a Junior Security Analyst
* Security monitoring and alert triage
* Escalation and incident handling procedures

### Skills Practiced

* Alert monitoring
* Incident identification
* Security event analysis
* Escalation processes

## Task 2: Digital Forensics and Incident Response (DFIR)

### Key Concepts Learned

* Importance of digital forensics in cybersecurity investigations
* Incident response lifecycle
* Evidence collection and preservation
* Containment, eradication, and recovery processes

### Skills Practiced

* Incident response fundamentals
* Digital evidence handling
* Security investigation processes

## Task 3: Threat Intelligence

### Key Concepts Learned

* Role of threat intelligence in cybersecurity operations
* Indicators of Compromise (IOCs)
* Threat actors and attack campaigns
* Intelligence gathering and analysis

### Skills Practiced

* IOC identification
* Threat intelligence analysis
* Threat monitoring

## Key Takeaways

A Junior Security Analyst serves as the first line of defence within a SOC by monitoring alerts, investigating suspicious activity, documenting findings, and escalating incidents when necessary. Effective incident response, digital forensics, and threat intelligence are essential components of modern cybersecurity operations.
