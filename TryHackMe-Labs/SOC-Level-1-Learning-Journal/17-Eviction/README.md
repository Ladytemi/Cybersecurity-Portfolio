# Eviction

## Overview

Eviction is a threat intelligence exercise focused on understanding adversary behavior using the MITRE ATT&CK Framework.

In this room, I investigated APT28 tactics, techniques, and procedures (TTPs) by navigating the MITRE ATT&CK Navigator and identifying attack techniques associated with different stages of the attack lifecycle.

The exercise reinforced how defenders can use threat intelligence and ATT&CK mappings to identify attacker behavior, improve detections, and understand adversary objectives.

## Topics Covered

- MITRE ATT&CK Framework
- ATT&CK Navigator
- Threat Intelligence
- APT28
- Initial Access
- Execution
- Persistence
- Defense Evasion
- Discovery
- Lateral Movement
- Collection
- Exfiltration

## Skills Demonstrated

- Threat Intelligence Analysis
- ATT&CK Mapping
- Adversary Profiling
- TTP Identification
- Security Operations Concepts

## Key Takeaways

- ATT&CK provides a common language for understanding attacker behavior.
- TTP-based detections are more valuable than relying solely on IOCs.
- Threat intelligence can be operationalized into detection opportunities.
- Understanding adversary behavior improves defensive capabilities.

## Outcome

Successfully identified and mapped APT28 techniques using the MITRE ATT&CK Framework.
