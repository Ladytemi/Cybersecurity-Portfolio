## Overview
The Pyramid of Pain is a threat intelligence framework that demonstrates how difficult it is for adversaries to change different indicators once defenders detect and block them.

The higher defenders move up the pyramid, the more operational cost and disruption is inflicted on attackers.

## Pyramid Layers

### Hash Values
File hashes such as MD5, SHA1, and SHA256 used to identify malicious files.

### IP Addresses
Network addresses used by attackers during operations.

### Domain Names
Domains registered and used by threat actors for phishing, malware delivery, and command-and-control activities.

### Network Artifacts
Observable indicators generated by malware or attacker communications, including user agents, URI patterns, and protocol usage.

### Tools
Malware, frameworks, and utilities used by attackers to conduct operations.

### Tactics, Techniques and Procedures (TTPs)
The behaviours, methodologies, and attack patterns used by adversaries. These are the most difficult indicators for attackers to change and therefore cause the greatest pain when detected.
