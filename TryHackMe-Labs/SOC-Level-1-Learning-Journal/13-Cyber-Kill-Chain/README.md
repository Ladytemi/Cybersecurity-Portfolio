## Overview

This room introduces the Cyber Kill Chain framework developed by Lockheed Martin.

The Cyber Kill Chain helps security teams understand how attackers progress through different stages of an attack, from reconnaissance to achieving their objectives. Understanding these stages helps defenders identify opportunities to detect, disrupt, and prevent cyber attacks.

## Cyber Kill Chain Phases

1. Reconnaissance
2. Weaponization
3. Delivery
4. Exploitation
5. Installation
6. Command and Control (C2)
7. Actions on Objectives

## Practical Exercise

A static-site challenge based on the 2013 Target data breach was completed by mapping attack activities to the correct Cyber Kill Chain phases.

Examples included:

- Spearphishing Attachment
- PowerShell
- Dynamic Linker Hijacking
- Exploit Public-Facing Application
- Fallback Channels
- Data from Local System
