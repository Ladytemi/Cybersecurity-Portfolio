# Investigating with ELK 101

## Overview

Completed a hands-on investigation lab using the Elastic Stack (ELK), focusing on log analysis, event correlation, threat investigation, and security monitoring through Kibana.

The room provided practical experience navigating Elasticsearch data, filtering logs, identifying suspicious activity, and analyzing security events within a SIEM environment.

## Skills Demonstrated

- Elastic Stack (ELK)
- Kibana Navigation
- SIEM Investigation
- Log Analysis
- Event Correlation
- Threat Hunting Fundamentals
- IOC Identification
- Security Monitoring
- Filtering and Searching Logs
- Incident Investigation

## Activities Performed

- Navigated Kibana dashboards and visualizations
- Queried and filtered log data
- Investigated security-related events
- Correlated events across multiple data sources
- Examined indicators of compromise (IOCs)
- Reviewed event details and metadata
- Identified suspicious activity through log analysis

## Key Learning Outcomes

- Learned how ELK can be used for centralized log management.
- Developed familiarity with Kibana search and filtering capabilities.
- Investigated security events using log data.
- Understood how analysts leverage ELK for threat detection and incident investigation.
- Improved practical SIEM investigation skills.

## Outcome

Successfully completed introductory ELK investigations and gained foundational experience using Elasticsearch and Kibana for security monitoring, event analysis, and threat detection workflows.
