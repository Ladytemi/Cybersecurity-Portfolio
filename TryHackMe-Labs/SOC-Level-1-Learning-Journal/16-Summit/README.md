# Summit

## Overview

Summit is a practical blue-team challenge that applies the Pyramid of Pain framework to real-world threat detection scenarios.

In this exercise, I worked as a Security Analyst supporting PicoSecure and was tasked with identifying and blocking multiple malware samples through progressively stronger detection mechanisms.

The challenge demonstrated how defenders can increase the operational cost for attackers by moving beyond simple indicators such as hashes and IP addresses and focusing on more resilient detection opportunities.

---

## Objectives

- Apply the Pyramid of Pain framework in a practical environment
- Investigate and identify malicious files
- Build increasingly effective detection logic
- Improve threat detection capabilities
- Understand how different indicators impact attacker operations

## Activities Performed

## Key Takeaways

- Simple indicators such as hashes are easy for attackers to change.
- Higher-level detections provide stronger defensive value.
- Detection engineering is essential for long-term security monitoring.
- Behavioral indicators are more resilient than basic IOCs.
- Effective defenders focus on increasing attacker operational costs.
