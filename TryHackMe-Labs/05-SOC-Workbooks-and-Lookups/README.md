# SOC Workbooks and Lookups

This room introduced SOC workbooks and lookup processes used by analysts during alert triage and investigations.

The exercises focused on building structured investigation workflows for suspicious emails, PowerShell activity, and network-based alerts. I learned how SOC analysts use workbooks to standardize investigations, gather evidence, document findings, and prepare escalation reports for higher-tier analysts.

Key concepts covered:

- Email investigation workflows
- PowerShell activity analysis
- Network investigation processes
- Asset inventory lookups
- Identity inventory lookups
- Evidence collection and triage
- Alert documentation and reporting
- Escalation procedures to L2 analysts

This room reinforced the importance of repeatable investigation procedures and proper documentation within Security Operations Centers.
