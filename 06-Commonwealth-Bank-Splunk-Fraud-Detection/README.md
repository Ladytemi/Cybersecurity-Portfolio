# Commonwealth Bank Cybersecurity Job Simulation

## Overview

Completed the Commonwealth Bank Introduction to Cybersecurity Job Simulation on Forage.

This simulation focused on core cybersecurity functions including data analysis, incident response, security awareness, and penetration testing.

## Tasks Completed

### Task 1 – Data Analysis

Analyzed cybersecurity data to identify suspicious activity patterns and trends. Reviewed dashboard information to support fraud detection and security monitoring activities.

### Task 2 – Incident Response

Investigated cybersecurity incidents and applied incident response principles to contain threats, assess impact, and support recovery efforts.

### Task 3 – Security Awareness

Created a password security awareness infographic to educate employees on password hygiene, multi-factor authentication, password managers, and password reuse prevention.

### Task 4 – Penetration Testing

Performed web application security testing activities to identify vulnerabilities, evaluate security weaknesses, and recommend remediation measures.
