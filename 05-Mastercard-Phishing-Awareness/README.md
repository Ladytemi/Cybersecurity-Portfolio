## Overview

Completed Mastercard's Cybersecurity Virtual Experience Program on Forage, focusing on phishing awareness, security awareness training, and phishing simulation analysis.

This simulation provided hands-on exposure to designing realistic phishing campaigns and interpreting phishing simulation results to improve employee cybersecurity awareness.

---

### Mastercard Cybersecurity Virtual Experience Program

**Focus Areas:** Phishing Awareness | Security Awareness Training | Social Engineering | Risk Reduction

Designed a realistic phishing email simulation and analysed phishing campaign results to identify risky employee behaviours. Applied phishing detection principles and security awareness strategies to improve organisational cyber resilience.

**Skills:** Security Awareness • Phishing Detection • Risk Assessment • Security Communication

---

## Tasks Completed

### Task 1 – Design a Phishing Email Simulation

Created a realistic phishing email designed to test employee awareness and identify users susceptible to phishing attacks.

Key objectives:

* Understand phishing techniques used by attackers
* Create a believable phishing email
* Apply social engineering concepts
* Increase phishing simulation effectiveness

### Task 2 – Interpret Phishing Simulation Results

Reviewed phishing simulation campaign metrics and identified employee behaviours that increased organisational risk.

Key objectives:

* Analyse phishing campaign results
* Identify high-risk behaviours
* Recommend security awareness improvements
* Support future phishing prevention training
