## Project Overview

This project is based on the AIG Shields Up: Cybersecurity Job Simulation completed on Forage in June 2026.

The simulation focused on cybersecurity threat analysis, zero-day vulnerability response, vulnerability remediation guidance, and ransomware decryption using Python.

## Scenario

The Cyber Defense Unit needed to respond to a reported zero-day vulnerability affecting enterprise systems. The simulation also included a ransomware-related technical task requiring a Python-based approach to bypass encryption by brute-forcing decryption keys.

## Tasks Completed

- Researched and analysed reported zero-day vulnerabilities
- Reviewed cybersecurity guidance and public vulnerability intelligence, including CISA publications
- Assessed vulnerability impact and remediation priorities
- Drafted a clear advisory email to guide affected teams on remediation steps
- Used Python to develop a brute-force decryption approach for a ransomware scenario
- Applied cybersecurity analysis and technical problem-solving to avoid ransom payment
